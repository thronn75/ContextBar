# Privacy Policy for ContextDock

Effective Date: January 1, 2026

ContextDock respects your privacy.

## Data Collection

ContextDock does NOT:

- Collect personal data
- Transmit data to external servers
- Use analytics services
- Track user behavior
- Store data in the cloud

All processing is done locally on your Mac.

## Local Data Storage

The app may store:

- Context preferences
- Pinned apps
- Project associations
- Local usage frequency (for smart suggestions)

This data remains on your device and is never transmitted.

## Accessibility Permission

If enabled, ContextDock uses macOS Accessibility APIs to:

- Read window titles
- Detect active projects
- Improve context detection

This information is processed locally and never leaves your Mac.

## Third-Party Services

ContextDock does not use third-party analytics, tracking, or advertising SDKs.
